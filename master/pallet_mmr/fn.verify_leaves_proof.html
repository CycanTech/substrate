Stateless MMR proof verification for batch of leaves.

This function can be used to verify received MMR primitives::BatchProof (proof) for given leaves set (leaves) against a known MMR root hash (root). Note, the leaves should be sorted such that corresponding leaves and leaf indices have the same position in both the leaves vector and the leaf_indices vector contained in the primitives::BatchProof.
